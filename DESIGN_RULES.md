# ChocoBrook Project: Design Standard Rulebook

## Standard Style: Toffeetown Whimsical 3D

All visual assets in the ChocoBrook ecosystem MUST adhere to the following rules:

### 1. Artistic Style
- **Semi-Realistic 3D Render**: The only allowed style.
- **Visual Aesthetic**: Pixar/Steampunk hybrid. rounded "toy-like" edges, volumetric lighting, and high-fidelity textures.
- **Materials**: Polished warm brass, glowing amber, clear glass, and weathered whimsical leather.

### 2. Forbidden Styles (Zero-Tolerance)
- **NO Photographic/Internet Images**: No real-world photos allowed.
- **NO Flat 2D Art**: No 2D illustrations or flat vector graphics.
- **NO Stock Imagery**: Everything must feel bespoke and created for the Toffeetown story.

### 3. Subject Matter
- **Investigation Stages**: Focus purely on mechanical objects and intricate parts.
- **Loveliness**: Every part must look "Lovely," rounded, and inviting, even if it is broken.

### 4. Implementation (Town Tycoon)
- **8 Stages**: Sequential investigation following mechanical logic (e.g., Mainspring -> Escapement -> Hub).
- **Zoom Glass**: High-detail views of these 3D objects.
